Simple Popup Widget Plugin lets you easily embed and promote anything on your website. You can easily promote any ads like adsence, amazon banner, facebook page etc.


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `simple-popup-widget` folder to the to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add Anything to this plugins from the widgets window in WordPress admin panel.
4. Configure the available options like background color, title, font size, popup time etc.
- Enjoy!
